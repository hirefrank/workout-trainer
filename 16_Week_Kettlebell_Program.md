# 16-Week Progressive Kettlebell Program

## Program Overview

**4-Day Weekly Split ‚Ä¢ Strategic Deloads ‚Ä¢ Progressive Overload**

### Bell Classifications
- **Moderate:** 35-45 lbs
- **Heavy:** 53-70 lbs
- **Very Heavy:** 88 lbs

### Program Phases
- Weeks 1-4: Foundation Phase (Moderate bells, build work capacity)
- Week 5: DELOAD
- Weeks 6-9: Strength Building Phase (Heavy bells, increase load)
- Week 10: DELOAD
- Weeks 11-14: Peak Performance Phase (Very heavy bells)
- Week 15: DELOAD
- Week 16: PEAK (Match December workouts)

### Training Rules
- All swing sets use even numbers only (6, 8, 10 sets)
- All reps use even numbers (8 or 10 reps per set)
- Rest 60-90 seconds between swing sets
- Rest 2-3 minutes between Turkish Getup sets
- Focus on explosive hip snap for swings
- Never rush Turkish Getups - maintain perfect form
